
# Akwaba Ashanti Fertility viewer
A simple 3d viewer for study.

<img src="./screenshot.png" alt="Akwaba Ashanti Fertility" width="75%">

## Getting Started

### Dependencies

* Threejs

### devDependencies
* Vite

### Installing

* Clone this repo
* Enter in the project folder

### Executing program

* Installing the program
```
npm install
```
* Running the program
```
npm run dev
```
